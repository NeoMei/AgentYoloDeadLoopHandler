# 🎉 AgentYoloDeadLoopHandler v1.0.0

## 🚀 首次正式发布

我们很高兴发布 AgentYoloDeadLoopHandler 的首个正式版本！这是一个专为 AI 开发工具设计的 YOLO 模式配置解决方案。

## ✨ 主要功能

### 🎯 核心特性
- **一键安装**: 简单的 bash 脚本，快速配置 YOLO 模式
- **双重支持**: 支持全局和项目级别的配置
- **自动备份**: 安装前自动备份现有配置
- **配置验证**: 内置验证功能确保配置正确
- **完全卸载**: 一键卸载功能，方便清理

### 🛠️ 支持的工具
- ✅ **Claude Code**: 完整配置和测试
- 🚧 **OpenCode**: 配置框架已准备
- 🚧 **Codex**: 配置框架已准备

### 🔧 安装方式

#### 方式一：一键安装脚本（推荐）
```bash
git clone https://github.com/NeoMei/AgentYoloDeadLoopHandler.git
cd AgentYoloDeadLoopHandler

# 全局安装
./setup-yolo-mode.sh --global

# 项目安装  
./setup-yolo-mode.sh --project
```

#### 方式二：手动配置
详见 [README.md](README.md#快速开始) 的详细说明
